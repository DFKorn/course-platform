# Course-Platform

Platform for selling courses

## Technical Stack

### Frontend
- Next.js 15
- TypeScript
- React 19
- Tailwind CSS v.4
- Shadcn
- Zod
- dnd kit

### Authentication
- Auth.js

### Database
- Neon Database
- Drizzle ORM (PostgreSQL with postgres.js driver)

### App Protection
- Arcjet
  
### Payments
- Stripe

### Deployment
- Vercel 
### Live - [Course-Platform](https://course-platform-sooty.vercel.app)
